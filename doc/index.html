jQuery.dForm

The jQuery.dForm plugin allows you to create your HTML forms programmatically from JavaScript objects (and JSON).

The plugin is pretty straightforward to use: 

			var data =
			{
				"action" : "index.html",
				"method" : "get",
				"elements" : 
				[
					{
						"name" : "textfield",
						"label" : "Label for textfield",
						"type" : "text",
						"value" : "Hello world"
					},
					{
						"type" : "submit",
						"value" : "Submit"
					}
				]			
			};
			$("#formid").buildForm(data);

Why would you need that?

Creating HTML forms is a very tedious task where you end up writing the same source over and over again. Many web frameworks will help you with creating your forms on the server side, but as soon as the client (e.g. client-side validation) comes into play it can become quite a mess with client and server concerns all mixed up together. Additionally you often just end up redefining your business objects as HTML forms which can be easily auto generated.

This plugin moves the generation of forms entirely on the client side where the server just has to provide a JavaScript object (usually as JSON) that contains all the information to create the form.

Through a neat subscription mechanism you can easily customize and extend the form creation process and add your own controls.
